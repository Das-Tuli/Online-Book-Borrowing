# ReadSphere

> A modern, elegant digital library management system featuring a classic editorial design system, secure authentication, dynamic borrowing, and real-time inventory tracking.

## 🌐 Live Site
[read-sphere-digital.vercel.app](https://read-sphere-digital.vercel.app)

## ⚡ Quick Start

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Set up `.env.local`:**
   ```env
   MONGODB_URI=your_mongodb_uri
   BETTER_AUTH_SECRET=your_better_auth_secret
   BETTER_AUTH_URL=http://localhost:3000
   NEXT_PUBLIC_APP_URL=http://localhost:3000
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS + DaisyUI
- **Database:** MongoDB + Mongoose
- **Auth:** BetterAuth
